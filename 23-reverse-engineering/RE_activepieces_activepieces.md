## Architecture

ActivePieces is an AI workflow automation platform with ~400 MCP servers for AI agents, enabling no-code/low-code automation.

## Key Design Patterns

1. **No-code/low-code** — Visual workflow builder
2. **MCP integration** — ~400 MCP servers available
3. **AI automation** — AI-powered workflow automation
4. **Self-hosted** — Can be deployed on own infrastructure
5. **Open source** — Fully open source with community contributions

## What We Can Learn

- No-code/low-code enables non-technical users
- MCP integration provides extensive tool coverage
- AI automation expands workflow capabilities
- Self-hosted deployment ensures data privacy
- Open source enables community contributions
