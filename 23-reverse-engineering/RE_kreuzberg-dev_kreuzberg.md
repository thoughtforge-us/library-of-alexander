## Architecture

Kreuzberg is a polyglot document intelligence framework with a Rust core for text, metadata, image, and structure extraction.

## Key Design Patterns

1. **Rust core** — High-performance document processing
2. **Polyglot support** — Multiple language bindings
3. **Multi-format** — PDF, DOCX, HTML, images, etc.
4. **Structure extraction** — Extract document structure
5. **Metadata extraction** — Extract document metadata

## What We Can Learn

- Rust core provides excellent performance
- Polyglot support enables wide adoption
- Multi-format support covers most use cases
- Structure extraction enables understanding
- Metadata extraction enables organization
