## Architecture

MCP-Use is a fullstack MCP framework for developing MCP Apps for ChatGPT/Claude and MCP Servers for AI Agents.

## Key Design Patterns

1. **Fullstack framework** — Both client and server support
2. **App development** — Build MCP apps for ChatGPT/Claude
3. **Server development** — Build MCP servers for agents
4. **Deployment tools** — Easy deployment to various platforms
5. **Template library** — Pre-built templates for common patterns

## What We Can Learn

- Fullstack framework simplifies development
- App development expands distribution channels
- Server development enables custom tools
- Deployment tools reduce friction
- Template library accelerates development

## Integration Ideas

- Use MCP-Use for MCP app development
- Build custom MCP servers for our tools
- Deploy to multiple platforms
- Use templates for common patterns
- Enable both client and server development
