## Architecture

LightAgent is a lightweight AI agent framework with memory, MCP, and skill support.

## Key Design Patterns

1. **Lightweight design** — Minimal dependencies and overhead
2. **Memory system** — Built-in memory for context retention
3. **MCP integration** — Model Context Protocol support
4. **Skill system** — Pluggable skill definitions
5. **Multi-agent collaboration** — Support for multiple agents working together

## What We Can Learn

- Lightweight design is easier to deploy and maintain
- Memory system is essential for context retention
- MCP integration enables tool standardization
- Skill system allows capability extension
- Multi-agent collaboration improves problem solving

## Integration Ideas

- Use lightweight design for Luna edge agent
- Implement memory system for cross-session context
- Add MCP integration for tool standardization
- Create skill system for capability extension
- Enable multi-agent collaboration across fleet
