## Architecture

Git MCP provides AI agents access to any GitHub repository's code, enabling code understanding and analysis.

## Key Design Patterns

1. **Remote access** — Access any GitHub repo without cloning
2. **Code search** — Search code across repositories
3. **File reading** — Read individual files
4. **Directory listing** — Browse repository structure
5. **Free tier** — No authentication required for public repos

## What We Can Learn

- Remote access eliminates local storage requirements
- Code search enables efficient code discovery
- File reading provides detailed analysis capability
- Directory browsing enables exploration
- Free tier drives adoption

## Integration Ideas

- Deploy Git MCP for code analysis
- Use remote access for efficient code lookup
- Enable code search for discovery
- Add file reading for detailed analysis
