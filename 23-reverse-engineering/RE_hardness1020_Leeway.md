## Architecture

Leeway is a workflow-driven AI agent framework that executes YAML-defined decision trees.

## Key Design Patterns

1. **YAML-defined workflows** — Workflows specified in YAML
2. **Decision trees** — Branching logic for different scenarios
3. **Declarative** — What to do, not how to do it
4. **Visualizable** — Workflows can be visualized as trees
5. **Version control friendly** — YAML is easy to diff and merge

## What We Can Learn

- YAML workflows are human-readable and editable
- Decision trees handle complex branching logic
- Declarative approach separates logic from implementation
- Visualization aids debugging and understanding
- Version control friendly enables collaboration

## Integration Ideas

- Use YAML for workflow definitions
- Implement decision trees for complex scenarios
- Create declarative agent specifications
- Add workflow visualization
- Store workflows in git for version control
