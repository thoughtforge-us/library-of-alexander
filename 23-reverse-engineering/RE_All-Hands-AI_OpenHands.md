## Architecture

OpenHands is an AI-powered software engineering agent that can write code, fix bugs, and complete development tasks.

## Key Design Patterns

1. **Software engineering focus** — Optimized for coding tasks
2. **Sandboxed execution** — Code runs in isolated environment
3. **File system access** — Agent can read and modify files
4. **Terminal access** — Agent can run shell commands
5. **Web browsing** — Agent can browse documentation

## What We Can Learn

- Software engineering focus attracts developers
- Sandboxed execution is essential for safety
- File system access enables concrete actions
- Terminal access expands agent capabilities
- Web browsing enables self-directed learning

- Deploy OpenHands on Nexus for coding tasks
