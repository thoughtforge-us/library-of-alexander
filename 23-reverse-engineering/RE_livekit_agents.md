## Architecture

LiveKit Agents is a framework for building real-time AI agents that can participate in voice and video calls.

## Key Design Patterns

1. **Real-time processing** — Low-latency audio/video processing
2. **Voice agents** — Agents that can speak and listen
3. **Video agents** — Agents that can see and be seen
4. **WebRTC** — Standard protocol for real-time communication
5. **Scalable** — Designed for production deployment

## What We Can Learn

- Real-time processing requires careful optimization
- Voice agents need low-latency TTS and STT
- Video agents add visual understanding
- WebRTC is the standard for real-time communication
- Scalability requires distributed architecture
