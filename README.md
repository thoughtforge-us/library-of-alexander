## 🤖 Automated Discovery Pipeline

This library is continuously updated by an automated pipeline:

1. **GitHub Search** — 40+ search queries in 11+ languages, every 4 hours
2. **Awesome List Scraping** — 10+ curated lists scraped and deduplicated
3. **Star Threshold** — Only tools with 30+ stars are added
4. **Deduplication** — Existing tools are skipped automatically
5. **Categorization** — Tools mapped to the correct category
6. **Translation** — Non-English descriptions translated to English
7. **Reverse Engineering** — Complex tools get architecture deep-dives

Run it yourself:
```bash
./scripts/discover_tools.sh
```
